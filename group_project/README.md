# Школьный портал - Объявления

Это Django-приложение для управления объявлениями школьного портала.

## Функции

- ✅ Объявления + категории
- ✅ Поиск и фильтр
- ✅ Роли (админ / ученик)
- ✅ Закрепленные объявления
- ✅ Дата окончания
- ✅ Красивый UI (Bootstrap)

## Установка и запуск

1. Установите зависимости (если нужно):
   ```
   pip install django
   ```

2. Примените миграции:
   ```
   python manage.py migrate
   ```

3. Создайте группы:
   ```
   python manage.py create_groups
   ```

4. Создайте тестовые данные:
   ```
   python manage.py create_sample_data
   ```

5. Запустите сервер:
   ```
   python manage.py runserver
   ```

## Доступ

- Админка: http://127.0.0.1:8000/admin/
  - Логин: admin
  - Пароль: admin123

- Объявления: http://127.0.0.1:8000/announcements/

## Роли

- **Admin**: Может создавать, редактировать и удалять объявления
- **Student**: Может только просматривать объявления

## Модели

- **Category**: Категории объявлений
- **Announcement**: Объявления с полями:
  - title: Заголовок
  - content: Содержание
  - category: Категория
  - author: Автор
  - created_at: Дата создания
  - updated_at: Дата обновления
  - end_date: Дата окончания
  - is_pinned: Закреплено ли объявление
